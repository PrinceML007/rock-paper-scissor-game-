
# Rock Paper Scissors Game using Python

## Project Description

This is a simple Python mini project based on the Rock Paper Scissors game.

The project is menu-driven and uses an infinite loop (`while True`) as required.

It has two playing modes:

1. Player vs Player
2. Player vs Computer

The game asks for player names, allows users to choose rounds from 1 to 5, and shows the winner after each round.

If the user chooses not to continue, the program exits with a Thank You message.

---

## Features

* Menu-driven program
* Infinite loop using `while True`
* Player vs Player mode
* Player vs Computer mode
* Player name input
* Round selection (1 to 5 rounds)
* Random computer choice using `random` module
* Winner announcement after every round
* Exit option with Thank You message
* Play again option
